### Install

1. Open Hyper™ preferences with your editor - Tip: `CMD+,` or at `~/.hyper.js`.
2. Add `hyper-kraftvaerk` to your plugins list.
3. Reload Hyper™ - `CMD+Shift+R`.

```js
plugins: ['hyper-kraftvaerk']
```
